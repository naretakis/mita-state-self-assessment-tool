# Archived On-Hold Specifications

This folder contains specifications for features that were planned but are currently on hold or have been deprioritized.

## Archive Policy

Specs are moved to this folder when:
- Feature was planned but not started
- Feature has been deprioritized
- Feature is waiting for external dependencies
- Feature has been superseded by other work

These specs may be reactivated in the future if priorities change or dependencies are resolved.
